# Task Overview

This repository contains my implementation for **Task 1 – Java Basics**. The objective of this task is to understand and apply the fundamental concepts of Java programming, including classes, objects, constructors, methods, user input handling, and basic arithmetic operations.

The project is a simple **Calculator Application** developed using Java. It allows users to enter two numbers and perform basic mathematical operations such as addition, subtraction, multiplication, and division.

This task helped me gain practical experience with object-oriented programming concepts and Java syntax.

---

# Project Description

The calculator program performs the following operations:

* Addition of two numbers
* Subtraction of two numbers
* Multiplication of two numbers
* Division of two numbers
* Handling division by zero errors

The program takes input from the user through the keyboard and displays the calculated results on the console.

---

# Concepts Used

### 1. Classes and Objects

A class named `Calculator` is created to define the calculator's properties and behavior.

An object of the Calculator class is created in the main method to access its functionality.

### 2. Constructor

A constructor is used to initialize the two numbers entered by the user.

```java
Calculator(int a, int b)
```

### 3. Methods

Separate methods are created for each arithmetic operation:

* `add()`
* `subtract()`
* `multiply()`
* `divide()`

### 4. User Input

The program uses the `Scanner` class to accept input from the user.

```java
Scanner sc = new Scanner(System.in);
```

### 5. Conditional Statements

An `if-else` statement is used to prevent division by zero.

---

# Program Workflow

### Step 1: Accept User Input

The user enters two integer values.

### Step 2: Create Calculator Object

The entered values are passed to the constructor.

### Step 3: Perform Operations

The calculator object calls the arithmetic methods.

### Step 4: Display Results

The results of all operations are printed on the console.
